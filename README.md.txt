# Event-Driven Pipeline on Render

This project is a simple event-driven web service built with **Flask** and deployed using **Render.com** on their free tier. It receives JSON data via a POST request and responds after logging the event.

---

## 🔧 Technologies Used
- Flask (Python)
- Gunicorn (Production WSGI server)
- Render (for web hosting)
- GitHub (CI/CD & version control)
